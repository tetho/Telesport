# Olympic Games App

## Description
Olympic Games App is a project for the TéléSport television channel. It is an interactive web application to prepare for the upcoming Olympic Games.

## Technologies Used
- Angular
- TypeScript
- HTML
- CSS
- Node.js
- ngx-charts

## Libraries and Tools
- **Angular CLI:** 18.0.6
- **Node.js:** 20.13.1
- **TypeScript:** 5.4.2
- **ngx-charts:** 20.5.0
